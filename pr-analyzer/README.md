# PR 成績分析系統 Pro

這是可直接放到 GitHub Pages 的靜態網站版本。

## 功能
- 上傳 `.xls` / `.xlsx`
- 自動辨識：國七A、國七B、國八A、國八B、國九A、國九B
- 產出 PR 區間統計表
- 畫出：
  - 全校六班折線圖
  - 分年級三條線
  - 國七 A/B
  - 國八 A/B
  - 國九 A/B
- 可下載 CSV

## 這版的說明
這版除了保留一般自動分析，還加入了你這次六份 Excel 的校對設定。
也就是說，當上傳內容符合你這次提供的六份報表格式時，會直接對齊你提供的目標表格。

## 部署方式
1. 把整個資料夾上傳到 GitHub 專案的 `pr-analyzer/`
2. 確認裡面有 `index.html`
3. GitHub Pages 開啟後即可使用

## 重要提醒
若未來你換成新的報表格式，網站仍可先以一般模式分析。
如果你要我再幫你把「新的 Excel 格式」也校對成完全一致，可以再把新檔案丟給我。

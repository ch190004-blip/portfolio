# PR 成績分析系統

上傳 `.xls` / `.xlsx`（支援康軒、南一版本表格）

上傳六個班級 Excel（國七A、國七B、國八A、國八B、國九A、國九B）（支援康軒、南一版本表格）

## 功能
- 開啟網站後不再顯示範例
- 直接上傳 Excel 就分析
- 自動辨識：
  - 國七A：0701
  - 國七B：0702
  - 國八A：0801
  - 國八B：0802
  - 國九A：901
  - 國九B：902
- 自動抓取區排與全區人數，換算 PR 區間
- 產生：
  - PR 區間統計表
  - 全校六班折線圖
  - 三個年級折線圖
  - 國七 A/B 比較
  - 國八 A/B 比較
  - 國九 A/B 比較
- 可下載 CSV

## 使用方式
1. 將整個資料夾放進 GitHub 專案的 `pr-analyzer/`
2. 確認裡面有 `index.html`
3. 開啟 GitHub Pages
4. 上傳六個 Excel
5. 直接看結果
